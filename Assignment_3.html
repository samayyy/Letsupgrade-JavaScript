Datatypes in HTML

Datatypes are classified into 2 types in javascript.

  • Primitive Datatype
  • Non-Primitive Datatype

Primitive Datatype

  1. Number

    2. A number data type can be an integer, a floating point value, an exponential value, a ‘NaN’ or a ‘Infinity’.

    Example

    var num=10;

    var dec=2.3;

    var exp=10e5;

  2. String

    3. The string data type in JavaScript can be any group of characters enclosed by a single or double-quotes or by backticks.

    Example

    var singleQuote='This is a String';

    var doubleQuote="This is a String";

  3. Boolean

    4. The boolean data type has only two values, true and false. It is mostly used to check a logical condition.

    Example

    var bool=true;

    var bool=false;

  4. Undefined

    5. Undefined data type means a variable that is not defined. The variable is declared but doesn’t contain any value.

    Example

    var a;

  5. Null

    6. The null in JavaScript is a data type that is represented by only one value, the ‘null’ itself. A null value means no value.

    Example

    var value=null;

  6. Symbol

    7. The ‘symbol’ data type is new in es6. It is one of the new features of es6. The symbol data type defines a property of an object which is private to the object. It refers to the ‘key’ of the key-value pair of an object.

    Example

    var object1 = { name: ‘Shalini’, age: 25, city: ‘Mumbai’ }; var occupation=Symbol(‘engineer’);

    The function Symbol() is used to create a new symbol. Here we have created a symbol ‘occupation’ with the value ‘engineer’ for the above object ‘object1’.

Non-Primitive Datatypes

  1. Object

    2. The ‘object’ is a non-primitive data type in JavaScript.

    Example

    var obj= { name: ‘Shalini’, age: 25, city: ‘Mumbai’ };

  2. Array

    3. An array in JavaScript is an object data type. An array contains more than one value with a numerical index, where the index starts from 0

    Example

    var array=[1,"string",bool,32.5];
