What is console?

The console object provides access to the browser's debugging console.It is a place where the output arrives when a particular code is executed

Methods in console

  1. console.assert()

    2. Log a message and stack trace to console if the first argument is false.

    Example

    console.assert(document.getElementById("demo"), "You have no element with ID 'demo'");

  2. console.clear()

    3. Clear the console.

    Example

    console.clear();

  3. console.count()

    4. Log the number of times this line has been called with the given label.

    Example

    for (i = 0; i < 10; i++) { console.count(); }

  4. console.countReset()

    5. Resets the value of the counter with the given label.

    Example

    for (i = 0; i < 10; i++) { console.count(); console.countReset(); }

  5. console.debug()

    6. Outputs a message to the console with the log level debug

    Example

    console.debug(obj_1 [, obj_2, ..., obj_N]);

  6. console.error()

    7. Outputs an error message. You may use string substitution and additional arguments with this method.

    Example

    console.error("This is an error");

  7. console.exception()

    8. An alias for error()

    Example

    console.exception("This is an exception");

  8. console.time()

    9. Starts a timer you can use to track how long an operation takes

    Example

    var a=10; console.time(a);

  9. console.log()

    10. For general output of logging information. You may use string substitution and additional arguments with this method.

    Example

    console.log("This is log function");

  10. console.warn

    11. Outputs a warning message to the Web Console.

    Example

    console.warning("This is a warning");
